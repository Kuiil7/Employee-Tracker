# Employee Tracker

A CLI CMS app that allows you to view and interact with information stored in a database. The app allows you to add, vew, and update departments, roles, employees. 

To start:
    node app.js

Created with: 
    Node.js
    Inquirer.js
    MySQL
    console.table
